# Dietitian Sajal - Clinical Nutritionist Website

![Website Preview](https://via.placeholder.com/1200x630/1e1e2f/c17b4c?text=Dietitian+Sajal+Website)

## ✨ **Features**

- **Modern Design:** Premium gradient themes, animated background orbs, and glassmorphism effects
- **Fully Responsive:** Optimized for all devices – mobile, tablet, laptop, and desktop
- **AI Chat Assistant:** Interactive chatbot to answer nutrition-related questions and guide visitors
- **WhatsApp Integration:** Direct booking and quick contact via WhatsApp with pre-filled messages
- **Dynamic Content:** Certificates, services, and blog posts can be updated via localStorage
- **SEO Optimized:** Meta tags, Open Graph tags, Twitter cards, and schema markup for better search visibility
- **High-Quality Icons:** Font Awesome 6 icons throughout for a professional look
- **Fast Loading:** Lightweight images and optimized code for quick page loads

---

## 🛠️ **Technologies Used**

- **HTML5** – Semantic markup for better SEO and accessibility
- **CSS3** – Custom styling with variables, animations, and responsive design
- **JavaScript (ES6+)** – Interactive features, form validation, and dynamic content
- **Font Awesome 6** – High-quality icons
- **AOS (Animate on Scroll)** – Scroll animations
- **Google Fonts** – Cormorant Garamond & Montserrat fonts
- **GitHub Pages** – Free hosting

| **Appointment (appointment.html)** | Booking form, contact info, working hours, location map, FAQ |
